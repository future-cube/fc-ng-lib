## Quickstart

在其他项目中，使用 `npm install @future-cube/theme --save` 安装类库

## Demo

+ 迁出 `git clone https://github.com/future-cube/fc-ng-lib.git`
+ 安装第三方类库 `yarn`
+ 启动 `npm run start`
